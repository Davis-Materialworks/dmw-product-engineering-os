# No Self-Improvement Loop

## Root Cause
Lessons from projects are not stored, so repeated failures recur.

## Detection Pattern
Final output has no lessons learned, missing rules, new patterns, or benchmark-change recommendations.

## Correction Strategy
Append project learning artifacts to knowledge/evolution after each substantial project.

## Prevention Rule
Every final major XD-to-code task must produce an evolution note or explicitly state why none is needed.
