# Doherty Threshold

## Principle
Productivity improves when systems respond within roughly 400ms for interactive feedback.

## Cite When
Cite when latency, streaming, filtering, search, or dashboard refresh behavior lacks feedback.

## XD Critique Pattern
- Identify the affected user decision or action.
- Explain the likely cognitive, motor, or trust cost.
- Recommend a design change that preserves intent while reducing friction.
