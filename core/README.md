# DMW Product Engineering OS Core

Core is the agent-independent intelligence layer. It contains durable knowledge, agents, benchmarks, patterns, governance, reviews, compliance, memory, and evolution.

Adapters may translate this intelligence for Codex, Claude Code, Cursor, and OpenCode, but adapters must not fork core rules or create divergent truths.

## Canonical Domains
- `knowledge/`
- `agents/`
- `benchmarks/`
- `patterns/`
- `governance/`
- `reviews/`
- `compliance/`
- `memory/`
- `evolution/`
