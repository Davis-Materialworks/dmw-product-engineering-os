# Miller's Law

## Principle
Working memory is limited; users struggle when too many items must be held at once.

## Cite When
Cite when forms, onboarding, dashboards, or comparison UIs overload memory.

## XD Critique Pattern
- Identify the affected user decision or action.
- Explain the likely cognitive, motor, or trust cost.
- Recommend a design change that preserves intent while reducing friction.
