# Human Override

## Purpose
Classify recommendations as Recommend, Strongly Recommend, Require, or Block. Use Block for unacceptable accessibility, security, compliance, or data-loss risk.

## Required Use
Consult this module when the project, recommendation, or review touches its topic.

- finding:
- evidence:
- recommendation:
- owner:
- override_level: Recommend | Strongly Recommend | Require | Block
