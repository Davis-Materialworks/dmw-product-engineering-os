# Mobile Architecture

## React Native

Use native primitives, safe areas, platform-aware gestures, accessibility labels, dynamic type, and Reanimated for complex motion.

## Flutter

Use widgets that reflect design-system tokens, theme extensions, semantics, adaptive layout, and golden tests.

## SwiftUI

Use semantic views, environment values, dynamic type, accessibility modifiers, and matchedGeometryEffect for XD-style continuity motion.

## Mobile review

Check thumb reach, touch target size, safe areas, offline/error states, permissions, platform conventions, and performance.
