# Quality Gates

## Minimum checks

- Type check
- Lint
- Unit/component tests where available
- Accessibility checks for changed UI
- Responsive manual review notes

## Recommended tests

- Component behavior tests for variants/states
- Visual regression for XD fidelity
- E2E tests for critical flows
- axe accessibility scans
- Keyboard navigation tests
- Form validation tests

## Acceptance statement

Every final response should say which checks ran and which were skipped.
