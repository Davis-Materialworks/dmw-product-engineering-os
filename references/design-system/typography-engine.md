# Typography Engine

## Extract

- font families
- weights
- sizes
- line heights
- letter spacing
- text transforms
- paragraph spacing
- heading hierarchy
- responsive scaling

## Heuristics

- Similar sizes within 1–2px may be the same token unless visually intentional.
- Headings should map to document structure, not just visual size.
- Body text below 14px is risky for enterprise and public-service UI.
- Long-form content needs readable line length.

## Output

Produce type tokens and component mappings: heading, body, caption, label, helper, code, display.
