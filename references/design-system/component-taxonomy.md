# Component Taxonomy

## Foundation

Button, IconButton, Link, Badge, Avatar, Tooltip, Popover, Dialog, Sheet, Tabs, Accordion, DropdownMenu, Breadcrumb, Pagination, Progress, Spinner, Skeleton, Toast.

## Forms

TextField, Textarea, Select, Combobox, Checkbox, RadioGroup, Switch, Slider, DatePicker, FileUpload, FormField, FieldError, FieldHint.

## Data

DataTable, CardGrid, FilterBar, SearchInput, SortControl, EmptyState, ErrorState, MetricCard, ChartPanel, Timeline, ActivityFeed.

## Enterprise

PermissionGate, AuditLog, RoleBadge, StatusBadge, ApprovalFlow, BulkActionBar, ExportMenu, DataRetentionNotice, SensitiveValue, IdentityCard.

## Industry extensions

Finance: TransactionRow, RiskBadge, KYCStep, AccountSummary.
Healthcare: PatientHeader, MedicationList, AppointmentCard, ClinicalAlert.
Education: CourseCard, AssignmentRow, GradeBadge.
Government: EligibilityStep, CaseStatus, ServiceNotice.
E-commerce: ProductCard, CartSummary, CheckoutStep.
