# Accessibility: WCAG AA/AAA Operating Rules

Target WCAG 2.2 AA by default. Flag AAA improvements where reasonable.

## Must check

- color contrast
- semantic HTML/native controls
- keyboard navigation
- focus order
- focus visibility
- screen reader names/descriptions
- error identification
- labels and instructions
- target size
- reduced motion
- reflow/responsive zoom
- language and reading order

## Forms

Every field needs a programmatic label, hint where needed, error message tied to the field, and validation that does not depend only on color.

## Dialogs

Dialogs require title, focus trap, escape behavior, restore focus, inert background, and semantic role.

## Data tables

Tables require captions or contextual headings, header scopes, sortable state announcements, and keyboard-accessible controls.
