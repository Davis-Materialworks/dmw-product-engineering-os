# Keyboard and Screen Reader Patterns

## Keyboard

- Tab order follows visual/task order.
- Enter/Space activate buttons appropriately.
- Escape closes dismissible overlays.
- Arrow keys operate menus, tabs, radios, sliders, and grids where expected.
- Focus is never hidden behind overlays.

## Screen reader

- Every interactive element has an accessible name.
- Decorative icons are hidden.
- Status updates use live regions sparingly.
- Errors are announced and linked.
- Loading states do not create noisy announcements.
